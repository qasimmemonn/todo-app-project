# Phase 1: Core Todo Console App

This is a foundation implementation of a command-line Todo application.

## Features
- Create tasks with titles and descriptions
- List all tasks with their current status
- Toggle task completion status
- Update existing task details
- Delete tasks

## Technical Stack
- **Language**: Python 3.11
- **Storage**: In-memory (Volatile)
- **Architecture**: Separated shared business logic (skills) and UI (core)

## Getting Started

### Prerequisites
- Python 3.11 or higher

### Installation
No installation required for Phase 1. Simply clone the repository and run the script.

### Usage
Run the application from the project root:
```bash
python phase-1-core/app.py
```

## Running Tests
Tests are located in the `tests/` directory and use `pytest`.

```bash
pytest
```
